About RBI Agent

The RBI (Research-Backtest-Iterate) Agent is an AI-powered system that automates the strategy development process:

  • Research Phase: Analyzes trading ideas from text, YouTube videos, or PDFs
  • Backtest Phase: Generates Python backtesting code using the backtesting.py library
  • Debug Phase: Automatically fixes technical issues and optimizes the strategy

Note: This is a research tool. Past performance does not guarantee future results. Always test strategies thoroughly before live trading.
